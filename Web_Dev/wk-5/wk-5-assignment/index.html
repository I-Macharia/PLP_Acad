Week Five Assignment

JavaScript Practice

This assignment is designed to help practice and demonstrate understanding of JavaScript fundamentals, including variables, conditionals, functions, loops, and DOM manipulation.

Please ensure that your code is well-commented and organized for clarity.

Files to Submit:

  • index.html — the main HTML file (this file)
  • styles.css — your CSS file for styling
  • script.js — your JavaScript file including:
    • Variable declarations and conditionals (Part 1)
    • At least 2 custom functions (Part 2)
    • At least 2 loop examples (Part 3)
    • At least 3 DOM interactions (Part 4)

Each part of the assignment should be clearly commented and organized.
